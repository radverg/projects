Makefile commands:
make 				Compiles the project
make run			Compiles the project and runs icp executable
make doxygen		Generates html documentation into doc/html
make clean			Cleans all files possibly generated by this makefile
make pack			Create package of this project
make qmake			Regenerates project Makefile using qmake if necessary

Implementation details of the application are described on the  main page of the documentation (in file main.cpp, index.html in generated html).
